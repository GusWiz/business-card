# Gustavo Hernandez — Business Card

A simple, single-page personal business card built with HTML and CSS to showcase Gustavo Hernandez's contact information, location, and future goal.

## Purpose

- Provide a digital business card.
- Demonstrate layout and styling skills with a small, clean project.
- Serve as a deployable static page that can be shared with employers and peers.

## Features

- Avatar image with accessible alt text.
- Two-column layout using Flexbox for avatar and information sections.
- Custom colors, borders, and a subtle box shadow for visual emphasis.
- Small, self-contained project (index.html, style.css, assets/).

## Skills Learned

- **HTML:** semantic structure, image markup, organizing content into reusable sections.
- **CSS:** Flexbox layout, box-shadow, border shorthand vs. side-specific borders, color usage, spacing and alignment.
- **Accessibility basics:** using `alt` text for images and clear text hierarchy.
- **Project organization:** separating assets and styles for a small static site.
- **Deployment (conceptual):** preparing a static site for hosting on GitHub Pages, Netlify, or Vercel.

## Live site:

- https://gustavo-hernandez-business-card.netlify.app/

## Customization

- Update the name and text in `index.html`.
- Replace the avatar image in `assets/img/` and update the `src` path if needed.
- Tweak colors, spacing, and layout in `style.css`.
